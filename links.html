생각톡 히어로 섹션 링크

1. Function (기본)
기본 히어로 섹션 - AI가 사고력을 망친다는 경고 메시지
Function 버전 보기
http://localhost:3000/?v=function&f=trial
2. Gangnam (강남)
대치동 상위 1% 엄마들이 갈아탄 AI 튜터
Gangnam 버전 보기
http://localhost:3000/?v=gangnam&f=trial
3. Grade (성적)
4등급 → 1등급, 스스로 생각하는 공부 치트키
Grade 버전 보기
http://localhost:3000/?v=grade&f=trial
4. Recognition (인정)
아끼면서도 성적 올리니 남편이 좋아해요
Recognition 버전 보기
http://localhost:3000/?v=recognition&f=trial

📌 참고사항

  • 각 링크는 v 파라미터로 히어로 섹션을 구분합니다
  • f=trial은 2주 체험 퍼널입니다 (상담 신청은 f=consult)
  • localhost:3000을 실제 도메인으로 변경하여 사용하세요
  • v=aicopy도 사용 가능합니다 (AI로 숙제 복붙하는 아이)
